## 時間結構

### 顯示範圍
- 每天從 **08:00** 開始，至 **20:00** 結束
- 底層以 **30 分鐘** 為最小單位，共 24 個半格（slot 0 ～ 23）
- 畫面以 **1 小時** 為主要列高；半小時事件佔半列高度

### Slot 對照表（部分）

| Slot 序號 | 時間區間 |
|-----------|---------|
| 0 | 08:00 – 08:30 |
| 1 | 08:30 – 09:00 |
| 2 | 09:00 – 09:30 |
| 3 | 09:30 – 10:00 |
| 4 | 10:00 – 10:30 |
| 5 | 10:30 – 11:00 |
| 6 | 11:00 – 11:30 |
| 7 | 11:30 – 12:00 |
| 8 | 12:00 – 12:30 |
| 9 | 12:30 – 13:00 |
| 10 | 13:00 – 13:30 |
| 11 | 13:30 – 14:00 |

| … | … |
| 23 | 19:30 – 20:00 |

> **換算公式**：`slotIndex = (小時 - 8) × 2 + (分鐘 / 30)`

## 區塊合併規則

- 底層資料以每半小時存儲（key 格式：`dayIndex_slotIndex`）
- **渲染時自動合併**：同一天內相鄰時段填入**相同內容**，視覺合併為一個大區塊
- 合併後區塊右下角顯示時長標籤（例如 `1h30分`、`2h`）
- 儲存或清除操作同步套用至區塊內**所有半格**

## 互動規則

### 密碼鎖
- 頁面右上角顯示「🔒 已鎖定」
- 點擊任意格子或「⚙ 固定行程設定」會先彈出密碼框
- 密碼：`0975000321`
- 驗證成功後本次瀏覽不需再輸入；點擊 🔓 徽章可手動重新鎖定

### 新增 / 編輯事項
1. 點擊任意格子（或已存在的事件區塊）
2. 輸入密碼解鎖（若尚未解鎖）
3. 彈窗內可修改：**日期**、**開始時間**、**結束時間**（30 分鐘精度）、**內容**
4. 點擊「儲存」完成

### 清除事項
- 點擊已填入的格子 → 點擊「清除」按鈕
- 清除後若為午休時段，自動還原為預設午休狀態
- 清除固定行程後，該格不會被規則自動補回

### 快捷鍵
- `Esc`：關閉彈窗
- `Ctrl / ⌘ + Enter`：快速儲存

## 固定行程設定面板

- 點擊右上角「⚙ 固定行程設定」進入（需密碼）
- 可新增、刪除、修改任意固定行程規則
- 每條規則設定：**名稱**、**適用星期**（複選）、**開始時間**、**結束時間**（30 分鐘精度）
- **儲存設定**時：已刪除或修改的規則對應格子立即從所有週次中清除
- 手動修改過內容的格子不受規則刪除影響

## 資料格式（localStorage `lyq_weekData_v2`）

```json
{
  "YYYY-MM-DD": {
    "events": {
      "dayIndex_slotIndex": "事項內容"
    },
    "lunchOverride": {
      "dayIndex_slotIndex": true
    },
    "userCleared": {
      "dayIndex_slotIndex": true
    }
  }
}
```

- `dayIndex`：0（週日）～ 6（週六）
- `slotIndex`：0（08:00）～ 23（19:30）
- `userCleared`：使用者明確清除過的格子，不被固定規則自動填回

固定規則存於 `lyq_fixedRules_v2`：

```json
[
  {
    "label": "團練－晨練",
    "days": [1, 2, 4, 5],
    "startSlot": 0,
    "endSlot": 2
  },
  {
    "label": "英文課",
    "days": [1, 3, 5],
    "startSlot": 4,
    "endSlot": 7
  }
]
```

## 備註

- 午休（12:00–14:00）為每日保留時段，**除非使用者特別指示，否則不安排其他行程**
- 所有行程（含固定行程）均可點擊自由修改日期、時間與內容
- 日後新增固定行程規則可透過「⚙ 固定行程設定」操作，不需修改程式碼
